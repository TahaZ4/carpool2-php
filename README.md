# PHP Authentication Demo

This repository demonstrates a basic authentication system implemented using PHP and MySQL.

## Features
- User signup with password hashing (`password_hash`)
- User login with password verification (`password_verify`)
- Session-based authentication
- Prepared statements to prevent SQL injection

## Notes
This implementation was developed and tested locally using XAMPP on macOS.
It serves as a proof-of-concept PHP backend authentication example.
